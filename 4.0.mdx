# Instatrace 4.0 Release

We're excited to announce the latest release of Instatrace, with a host of new features and enhancements to help you manage your workflows more efficiently.

### Features

- Workflow Tracing: Detailed tracing capabilities for workflows, enabling you to see the exact path taken by a workflow and the status of each step along the way.
- Conditional States: Define branching logic within your state machines to handle different scenarios based on conditions.
- Custom Metrics: Define your own metrics for tracking and monitoring workflows.

### Improvements

- Visual Workflow Creation: New drag-and-drop capabilities and improved support for large workflows.
- Workflow Execution: Optimized for faster and more reliable processing, with better handling of error conditions and improved retry logic.
- User Interface: Updated with new icons and improved navigation for easier and more intuitive use.

### Bug Fixes

- Several bugs have been fixed in this release, including issues with state machine transitions and visual workflow display.

### Upgrade Instructions

- This release includes no breaking changes. To upgrade, simply follow the instructions provided in the documentation.
